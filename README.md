esse projeto se baseia em um website criado puramente por front-end, um grupo iniciante criou isto usando a ferramenta visual studio code para um trabalho de faculdade.
O projeto é um website comparativo, onde comparamos eletronicos digitais para a pessoa que quer se informar do que esta comprando, se é ruim ou bom, esse site ajudará essa pessoa a se guiar em qual opçao que ela esta em duvida ir
o projeto foi feito por 5 iniciantes, primeiro website dos cincos integrantes.
